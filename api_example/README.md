# API Example

## Описание
Скрипт `api_reader.py` загружает данные из [Cat Facts API](https://catfact.ninja/facts)
и сохраняет их в таблицу Pandas и CSV файл.

##  Как запустить
После запуска команды:
```bash
python api_reader_utf8.py
```
